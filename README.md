## Resume Match backend setup

- Local development: run `npm run dev:full` (frontend + backend together).
- Deployed frontend: set `VITE_API_BASE_URL` to your backend origin.

Example `.env`:

VITE_API_BASE_URL=https://your-backend-domain.com
